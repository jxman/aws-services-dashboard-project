# Screenshots Directory

This directory contains screenshots for the README.md file.

## Required Screenshots

Add the following screenshots once the dashboard is operational:

1. **dashboard-overview.png** (1200x630px recommended)
   - Main dashboard view showing key metrics
   - Include summary statistics and overview charts
   - Show navigation and overall layout

2. **regions-view.png** (1200x630px recommended)
   - Regional distribution visualization
   - Show resource breakdown by AWS region
   - Include regional charts/graphs

3. **services-view.png** (1200x630px recommended)
   - AWS services breakdown view
   - Show service utilization and metrics
   - Include service-specific visualizations

4. **coverage-view.png** (1200x630px recommended)
   - Account and service coverage metrics
   - Show coverage percentages and gaps
   - Include coverage analytics and insights

## Screenshot Guidelines

- Use consistent browser window size
- Capture at high resolution (Retina/2x if possible)
- Use production data (anonymize sensitive info)
- Ensure good contrast and readability
- Compress images for web (keep under 500KB each)
- Use PNG format for clarity

## Tools for Screenshots

- macOS: Cmd+Shift+4 (select area)
- Chrome DevTools: Device emulation for consistent sizing
- CleanShot X: Professional screenshot tool (optional)
- ImageOptim: Compress images without quality loss
